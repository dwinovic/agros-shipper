## Getting Started

### Prerequisites

This is an example of how to list things you need to use the software and how to install them.
* [Node Js](https://nodejs.org/en/download/)

### Installation

1. Clone This Repo
```sh
git clone https://github.com/dwinovic/agros-shipper
```
2. Go To Folder Repo
```sh
cd agros-shipper
```
3. Install Module
```sh
npm install
```
4. Start website
```sh
npm start
```
